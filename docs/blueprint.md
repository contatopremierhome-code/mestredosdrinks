# **App Name**: Mixology Master

## Core Features:

- Recipe Display: Showcase 500+ professional cocktail recipes.
- Recipe Teaser: Display horizontally scrolling recipe cards with attractive visuals.
- Pricing Plans: Offer dual pricing plans with clear distinctions.
- Bonus Showcase: Highlight exclusive bonuses for the premium plan.
- FAQ Section: Provide an accordion-style FAQ section.
- Sticky CTA: Implement a floating bottom bar CTA for mobile.

## Style Guidelines:

- Base color: Dark Mode base (#0F172A) to ensure colors pop.
- Primary color: Neon Lime (#BEF264) for a fresh, vibrant look.
- Secondary color: Electric Orange (#FB923C) and Tropical Magenta (#D946EF).
- Effects: Use Glassmorphism (semi-transparent blurred backgrounds).
- Headline font: 'Inter' (sans-serif) for a bold, modern feel.
- Body font: Clean and readable sans-serif font for easy reading.
- Thumb-friendly: 100% optimized for mobile with large buttons (min 48px height).
- Structure: Single-column layout for mobile optimization.
